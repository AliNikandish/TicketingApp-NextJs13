## Installation

#### Clone the project

```bash
  git clone https://github.com/AliNikandish/TicketingApp-NextJs13.git
```

#### Go to the project directory

```bash
  cd TicketingApp-NextJs13
```

#### Install TicketingApp with npm

```bash
  npm install
```
#### Run:
```bash
  npm run dev
```
